# LiveWall Pro - Wallpaper Repository

This repository contains the video wallpapers used by **LiveWall Pro**, a native macOS live wallpaper application.

## How to use

1. Open **LiveWall Pro** on your Mac
2. Go to the **Explore** tab
3. The repository [LiveWallPro](https://github.com/javiermmdev/LiveWallPro) is pre-configured
4. Click **Fetch** to browse available wallpapers
5. Download individually or click **Download All**

## Repository structure

Wallpapers must be organized in category folders with `.mp4`, `.m4v`, or `.mov` files:

```
├── Nature/
├── Space/
├── Anime/
├── Cars/
├── City/
├── Video Games/
├── Sci-fi/
├── Fantasy/
└── Animals/
```

Files placed outside these folders will appear without a category.

## Requirements

- **LiveWall Pro** for macOS 26+
- Video files in MP4/M4V/MOV format

# LiveWall Pro - Repositorio de Fondos

Este repositorio contiene los fondos de pantalla en vídeo utilizados por **LiveWall Pro**, una aplicación nativa de fondos de pantalla animados para macOS.

## Cómo usar

1. Abre **LiveWall Pro** en tu Mac
2. Ve a la pestaña **Explorar**
3. El repositorio `javiermmdev/livewallpapers-pro` viene preconfigurado
4. Pulsa **Obtener** para ver los fondos disponibles
5. Descarga individualmente o pulsa **Descargar Todo**

## Estructura del repositorio

Los fondos deben estar organizados en carpetas de categoría con archivos `.mp4`, `.m4v` o `.mov`:

```
├── Nature/
├── Space/
├── Anime/
├── Cars/
├── City/
├── Video Games/
├── Sci-fi/
├── Fantasy/
└── Animals/
```

Los archivos fuera de estas carpetas aparecerán sin categoría.

## Requisitos

- **LiveWall Pro** para macOS 26+
- Archivos de vídeo en formato MP4/M4V/MOV
